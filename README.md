# 📖 Math Magicians <a name="about-project"></a>


**Math Magicians** "Math magicians" is a website for all fans of mathematics. It is a Single Page App (SPA) that allows users to:

- Make simple calculations.
- Read a random math-related quote.

By building this application, using React.

### Prerequisites

In order to run this project you need:

<ul>
<li>Visual Code Studio</li>
<li>Git Bash</li>
<li>Node.js</li>
</ul>

### Setup

Clone this repository to your desired folder:

-sh
  cd (your folder)
  https://github.com/Danfall369/math-magicians.git

### Install

Install this project with:

To install this project, you need to run: ``` npm install ```

### Usage

To run the project, execute the following command: ``` npm start ```

<p>-With Visual Studio Code, open the folder where you downloaded this repository</p>
